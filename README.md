## Build from source
### Requirements
-   [Node.js](https://nodejs.org/)
-   [Rust](https://rust-lang.org/)
-   [Tauri](https://tauri.app/)
-   [pnpm](https://pnpm.io/)

1. Clone repository:
```zsh
git clone https://github.com/userift/rift.git
cd Rift
```
2. Install dependencies:
```zsh
pnpm i
tauri build
```
3. Run application
```zsh
tauri run
```
4. Build application
```zsh
tauri build
```
